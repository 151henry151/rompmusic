# RompMusic Client Specification

Based on the initial specification prompt, the app must include:

## Visual/UI Elements
- [x] Artist and album pictures/artwork
- [x] Playback bar showing progress through track
- [x] Time indicators: elapsed and total track length
- [x] Previous track button in player
- [x] Next track button in player
- [x] Volume adjuster in player

## Settings Page
- [x] All expected settings for a music player (Account, Playback, About, etc.)

## Home Page
- [x] "Recently played" section
- [x] "Recently added" section  
- [x] "Most played" section
- [x] Other expected elements of a modern music player

## Track Interaction
- [x] Clicking a track (without clicking play button) opens track detail page
- [x] Track detail page shows track information
- [x] Track detail page has "View album" button
- [x] "View album" navigates to album with track list
- [x] The track navigated from is highlighted in the album list

## Search
- [x] Search bar searches album names
- [x] Search bar searches artist names
- [x] Search bar searches track names
